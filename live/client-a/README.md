# Stacks: client-a

Este diretorio contem as stacks reais de infraestrutura para client-a em tres ambientes:
- dev
- stg
- prod

Cada ambiente possui os stacks:
- vpc
- rds
- ec2
- ec2-autoscaling
- ecs
- eks
- iam
- security-baseline
- s3

Cada stack possui README local com detalhes tecnicos, dependencias e fluxo de execucao.
