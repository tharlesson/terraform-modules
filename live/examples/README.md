# Examples

Este diretorio contem exemplos executaveis para client-a-dev:
- client-a-dev-vpc
- client-a-dev-rds
- client-a-dev-ec2
- client-a-dev-ec2-autoscaling
- client-a-dev-ecs
- client-a-dev-eks
- client-a-dev-iam
- client-a-dev-security-baseline
- client-a-dev-s3

Use esses exemplos como ponto de partida para novos clientes e novos ambientes.
No estado atual, os exemplos publicados sao somente para dev (client-a-dev-*).
Para stg/prod, a promocao recomendada e copiar o exemplo dev e ajustar terraform.tfvars/backend.hcl.
Cada exemplo possui README local com orientacoes de adaptacao.
